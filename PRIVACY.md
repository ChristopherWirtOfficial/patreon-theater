# Privacy Policy — Theater Mode for Patreon

_Last updated: 2026-06-21_

**Theater Mode for Patreon does not collect, store, transmit, or share any
data.**

- It does not gather personal information, browsing history, or analytics.
- It makes no network requests of its own.
- It uses no cookies, trackers, or remote/hosted code — all code ships inside
  the extension package.
- It runs only when you click the toolbar icon on a Patreon page, and only
  rearranges the existing video element on that page (it does not read page
  content for any other purpose).

Because no data is collected, none is sold, transferred, or used for any
purpose.

If you have questions, open an issue at
https://github.com/ChristopherWirtOfficial/patreon-theater
